# Godwa Frontend - Environment Setup

# 1. Create a file named .env.local in this directory
# 2. Paste the following line into it:
# NEXT_PUBLIC_GOOGLE_CLIENT_ID=your_client_id_here.apps.googleusercontent.com

# To get your Google Client ID:
# 1. Visit: https://console.cloud.google.com/
# 2. Project > Create/Select
# 3. APIs & Services > Credentials
# 4. Create Credentials > OAuth client ID (Web Application)
# 5. Add Authorized Origin: http://localhost:3000
# 6. Copy the resulting Client ID
